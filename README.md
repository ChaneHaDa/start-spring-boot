# start-spring-boot

## 목적
해당 프로젝트는 Spring Boot를 활용하여 RESTful API를 구현하는 프로제트

## Environment
* JAVA 17
* Gradle-Froovy
* Spring Boot 3.25

## 설명
restful API를 구축하는 과정을 버전별로 관리하고 있음

## 내용
* JPA, RestController, Entity Valid
* API 버전관리
* HATEOAS, HAL explorer, Filtering
* Spring Security(기본 인증, CORS, CSRF, JWT, Oauth2)
* AOP
